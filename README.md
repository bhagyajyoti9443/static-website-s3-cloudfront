## 🚀 Static Website Deployment on AWS

This project demonstrates deploying a static website using Amazon S3 and Amazon CloudFront.

### 🌐 Live Demo
https://d2eia1kb7ysdhx.cloudfront.net/index.html

### 🛠 Services Used
- Amazon S3 (Static website storage)
- Amazon CloudFront (CDN)
- Origin Access Control (OAC)
- AWS WAF (Monitor mode)

- index.html
